## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio-project.git
   cd portfolio-project
   ```
2. Open `index.html` in your browser, or use the VS Code "Live Server" extension for auto-reload while editing.

## Deployment

This site is static, so it can be hosted for free with any of the following:

- **GitHub Pages**: Settings → Pages → set source to `main` branch.
- **Netlify**: drag and drop the folder, or connect the repo.
- **Vercel**: import the repo and deploy.
